  ```sh
  Please use python version 3.5 >
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jmateocarbon/hybrid-analysis-api.git
   ```
2. Install python packages
   ```sh
   pip install -r requirements.txt
   ```

Just run the script HybridAnalysis.py
